# 🤖 IoT Powered Industrial Robot

An intelligent **ESP32-based Industrial Robot** designed for smart industrial automation, real-time hazard detection, and remote monitoring. The robot integrates **ESP32, ESP32-CAM, fire sensor, gas sensor, ultrasonic sensor, LCD display, buzzer, and DC motors** to autonomously monitor industrial environments, avoid obstacles, detect emergencies, and transmit live data through IoT platforms. This project enhances workplace safety, minimizes human intervention in hazardous areas, and provides a scalable solution for Industry 4.0 applications.

---

## 🚀 Features

- 🔥 Fire Detection and Alert System
- ☁️ Hazardous Gas Leak Detection
- 📷 Live Video Streaming using ESP32-CAM
- 📏 Ultrasonic Sensor-based Obstacle Avoidance
- 📡 Real-Time IoT Monitoring
- 📺 LCD Display for Live Status
- 🔊 Emergency Buzzer Alerts
- 🤖 Autonomous Robot Navigation
- ⚡ ESP32-based Low Power Embedded System

## ⚙️ Working

1. ESP32 continuously reads sensor data.
2. Fire and gas sensors monitor hazardous conditions.
3. Ultrasonic sensor detects obstacles and avoids collisions.
4. ESP32-CAM provides live video streaming.
5. LCD displays system status.
6. Buzzer alerts when hazards are detected.
7. Sensor data is uploaded to the IoT cloud for remote monitoring.
